# LangChain Demo

这是一个用 Flask + LangChain 搭建的简单对话演示项目。

## 功能
- 在网页上提供一个输入框
- 用户输入问题后，返回一个简单的回复
- 可以扩展接入 LangChain，让模型生成更智能的回答

## 使用方法
1. 打开网页后，在输入框里输入你的问题。
2. 点击提交，页面会显示模型的回复。

## 文件说明
- `app.py`：主程序，包含 Flask 代码。
- `README.md`：项目说明文件。

## 演示目的
这个项目主要用于展示如何在 GitHub 上创建一个简单的对话应用，并证明你会调用和使用 LangChain。
